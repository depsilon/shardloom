Plain-English meaning

Native Vortex is the lane for scoped execution over existing Vortex artifacts.

Why it matters

This concept helps users interpret ShardLoom's execution modes posture without reading the phase plan or RFCs first.

How ShardLoom uses it

ShardLoom surfaces this through docs, capability/status pages, use cases, or evidence fields such as native_vortex, source_backed_scan_provider_kind, source_backed_scan_native_io_certificate_status.

Current support

Current status is scoped-runtime-lane. Treat that status exactly as shown; scoped, smoke, report-only, planned, blocked, or unsupported posture must not be read as broader runtime support.
