Plain-English meaning

Foundry dev-stack smoke is a local/style-only proof path that keeps real Foundry runtime and compute invocation disabled.

Why it matters

This concept helps users interpret ShardLoom's platform boundaries posture without reading the phase plan or RFCs first.

How ShardLoom uses it

ShardLoom surfaces this through docs, capability/status pages, use cases, or evidence fields such as foundry_runtime_invoked, foundry_compute_invoked, foundry_spark_invoked, public_foundry_claim_allowed.

Current support

Current status is local-style-proof. Treat that status exactly as shown; scoped, smoke, report-only, planned, blocked, or unsupported posture must not be read as broader runtime support.

What it does not claim
