Python front door

The first Python smoke exercises the scoped local read/query path and prints the evidence fields that bound the claim. Source-checkout and benchmark examples may add explicit schemas, repo roots, and build profile order for reproducibility.

Python
import shardloom as sl

ctx = sl.context()
result = (
    ctx.read("data/orders.csv")
       .filter(sl.col("status") == "paid")
       .limit(10)
       .collect()
)

print(result.output_row_count)
print(result.first_result_row)
print(result.claim_summary.claim_gate_status)
print(result.fallback_attempted, result.external_engine_invoked)
