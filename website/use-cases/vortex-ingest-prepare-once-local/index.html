Use Case

Vortex ingest prepare-once local smoke

Feature-gated local prepare-once fixture smoke over flat non-null int/uint/float/UTF-8/date32/timestamp rows only; default builds block deterministically. Certification depth is explicit: ingest_minimal records artifact bytes/digest and writer evidence without reopen proof, ingest_certified reopens/scans for row-count proof, and ingest_full_replay is blocked until downstream output replay evidence exists. No broad Vortex writer, object-store/table sink, production SQL/DataFrame, performance, superiority, Foundry, package-publication, or Spark-replacement claim.

smoke_supported

User goal

user who wants to create a local VortexPreparedState from an admitted local source

Execution mode

vortex_ingest_to_prepared_vortex

Engine mode

batch

Inputs / outputs

local_csv_flat_non_null_scalars, local_json_flat_non_null_scalars, local_jsonl_flat_non_null_scalars, local_parquet_when_features_enabled -> local_vortex_artifact, vortex_prepared_state_evidence

Runnable example or blocker

New-Item -ItemType Directory -Force target | Out-Null; "id,label,amount`n1,alpha,8`n2,beta,15`n" | Set-Content -Encoding utf8 target\vortex-ingest-source.csv; cargo run -q -p shardloom-cli --features vortex-write -- vortex-ingest-smoke target\vortex-ingest-source.csv target\vortex-ingest-source.vortex --allow-overwrite --format json; $env:PYTHONPATH = "python\src"; python -c "from shardloom import context; ctx=context(repo_root='.', profile_order=('debug','release')); r=ctx.prepare_vortex('target/vortex-ingest-source.csv','target/vortex-ingest-source.vortex', allow_overwrite=True); print(r.vortex_ingest_status, r.prepared_state_created, r.input_row_count, r.fallback_attempted, r.external_engine_invoked)"

Expected output/evidence

A local .vortex artifact plus VortexPreparedState id/digest, certification-level/status fields, writer evidence, optional reopen row-count proof for ingest_certified, source-state fields, and no-fallback/no-external-engine evidence.

Evidence fields

  • source_adapter_id
  • source_state_id
  • source_state_digest
  • ingress_route=vortex_ingest
  • vortex_ingest_status
  • prepared_state_id
  • prepared_state_digest
  • vortex_artifact_digest
  • writer_row_count
  • reopen_row_count
  • reopen_verification_status
  • certification_level
  • certification_status
  • timing_scope=ingest_only
  • fallback_attempted=false
  • external_engine_invoked=false
  • claim_gate_status
