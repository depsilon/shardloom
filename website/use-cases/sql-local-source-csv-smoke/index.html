Use Case

SQL local source projection/optional-filter/IN/limit, aggregate, group-by, top-N, and join smoke

Scoped local CSV, flat JSON/JSONL/NDJSON, and feature-gated flat scalar Parquet/Arrow IPC/Avro/ORC SELECT projection/filter/limit with admitted comparison, cast, numeric/string-transform predicate/projection, date, IN/BETWEEN, null/string/logical, LIKE, UTF-8 LOWER/UPPER/TRIM, and balanced-parentheses leaves; scalar aggregate, one-column group-by aggregate, single-key numeric top-N; ctx.sql collect/write; local JSONL/CSV and feature-gated flat scalar structured sinks; and one Python local CSV inner equi-join bridge. Arithmetic computed columns admit only <column> (+|-|*|/) <numeric-literal> AS <column>; string computed columns admit LOWER/UPPER/TRIM(<column>) AS <column>; lossy coercion, division by zero, and unsupported shapes block. IN/NOT IN admits up to 32 same-family literals including NULL with SQL three-valued WHERE-filter semantics. No broad SQL/DataFrame or production runtime, broad format fidelity, fanout, object-store/table source, generalized projections, joins/groups/orderings, timestamp/timezone or locale/collation completeness, arbitrary predicate-tree completeness, fallback, or performance claim.

Runnable example or blocker

New-Item -ItemType Directory -Force target | Out-Null; "id,customer_id,amount`n1,10,8`n2,20,15`n3,30,21`n4,99,13`n" | Set-Content -Encoding utf8 target\sql-local-source-join-fact.csv; "customer_id,segment`n10,seed`n20,enterprise`n30,startup`n" | Set-Content -Encoding utf8 target\sql-local-source-join-dim.csv; cargo run -q -p shardloom-cli -- sql-local-source-smoke "SELECT f.id,d.segment FROM 'target/sql-local-source-join-fact.csv' AS f INNER JOIN 'target/sql-local-source-join-dim.csv' AS d ON f.customer_id = d.customer_id WHERE f.amount >= 10 LIMIT 10" --format json; $env:PYTHONPATH = "python\src"; python -c "from shardloom import context; ctx=context(repo_root='.', profile_order=('debug','release')); r=ctx.read_csv('target/sql-local-source-join-fact.csv').join(ctx.read_csv('target/sql-local-source-join-dim.csv'), on='customer_id').select('f.id','d.segment').filter('f.amount >= 10').limit(10).collect(); print(r.output_row_count, r.join_runtime_execution, r.fallback_attempted, r.external_engine_invoked)"
