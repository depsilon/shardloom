Use Case

Python local file query-builder projection, preview, literal-column, aggregate, group-by, join, and top-N smoke

Scoped Python read_csv/read_json/read_parquet/read_arrow_ipc/read_avro/read_orc local-source smokes cover projection/filter/limit, preview/head/take, literal, cast, numeric arithmetic, UTF-8 lower/upper/trim, and Date32/UTC timestamp extract with_column(...), count, scalar aggregate, one-column group_by aggregate, single-key top-N, and one local CSV inner equi-join bridge over admitted local CSV, flat JSON/JSONL/NDJSON, and feature-gated flat scalar Parquet/Arrow IPC/Avro/ORC. Filters support admitted comparison, between/not-between, cast, numeric arithmetic with exact int64/float64 mixed coercion, date, IN/NOT IN, null/string/logical, LIKE/NOT LIKE, UTF-8 lower/upper/trim, and balanced-parentheses leaves. Local JSONL, CSV, and feature-gated flat scalar structured writes are admitted. No nested JSON, JSONPath, broader structured-format fidelity, pandas/Polars backend, broad DataFrame runtime, generalized expression with_column, generalized joins/groups/orderings, timezone-database or locale/collation completeness, subquery-backed IN/NOT IN, Vortex output, fanout, production SQL, object-store/table source, fallback, or performance claim.

Runnable example or blocker

New-Item -ItemType Directory -Force target | Out-Null; "id,label,amount`n1,alpha,8`n2,beta,15`n3,beta,21`n4,gamma,`n" | Set-Content -Encoding utf8 target\sql-local-source-smoke.csv; $env:PYTHONPATH = "python\src"; python -c "import shardloom as sl; ctx=sl.context(repo_root='.', profile_order=('debug','release')); preview=ctx.read_csv('target/sql-local-source-smoke.csv').preview(limit=2); head=ctx.read_csv('target/sql-local-source-smoke.csv').head(limit=2); take=ctx.read_csv('target/sql-local-source-smoke.csv').take(2); workflow=ctx.read_csv('target/sql-local-source-smoke.csv').select('id','label').where(sl.col('amount').between(10, 30)).limit(1); r=workflow.write('target/sql-local-source-result.jsonl', allow_overwrite=True); c=workflow.write_csv('target/sql-local-source-result.csv', allow_overwrite=True); w=ctx.read_csv('target/sql-local-source-smoke.csv').select('id','label').with_column('batch_id', 1).filter(sl.col('amount') >= 10).limit(10).collect(); p=ctx.read_csv('target/sql-local-source-smoke.csv').select('id').with_column('adjusted', sl.col('amount') + 5).filter(sl.col('amount') >= 10).limit(10).collect(); n=ctx.read_csv('target/sql-local-source-smoke.csv').filter(sl.col('amount') >= 10).count(); a=ctx.read_csv('target/sql-local-source-smoke.csv').aggregate('count(*)','sum(amount)','avg(amount)').limit(1).collect(); g=ctx.read_csv('target/sql-local-source-smoke.csv').group_by('label').agg('count(*)','sum(amount)').limit(10).collect(); t=ctx.read_csv('target/sql-local-source-smoke.csv').select('id','label').sort('amount', descending=True).limit(2).collect(); print(preview.output_row_count, head.output_row_count, take.output_row_count, r.output_path, r.output_native_io_certificate_status, c.output_path, c.output_format, c.output_native_io_certificate_status, w.envelope.field('literal_projection_columns'), p.numeric_arithmetic_projection_operator, p.numeric_arithmetic_projection_output_columns, n.aggregate_functions, a.aggregate_operator_family, g.aggregate_operator_family, g.group_by_columns, t.sort_keys, t.top_n_limit, r.fallback_attempted, r.external_engine_invoked)"
