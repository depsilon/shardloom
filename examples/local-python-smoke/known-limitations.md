# Known Limitations

- This example proves import, CLI resolution, status, smoke, capabilities, and no-fallback policy.
- It does not read or write datasets.
- It does not emit execution or Native I/O certificates because no data path is executed.
- It does not prove PyPI, Conda, Foundry, object-store, SQL, DataFrame, live, or hybrid support.
- It does not install benchmark comparison engines or use them as fallback execution.
