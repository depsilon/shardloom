# Contributing to ShardLoom

ShardLoom is an Apache-2.0 licensed open-source project.

## Contribution principles

Contributions should be:

- Original work
- Tested
- Documented where appropriate
- Compatible with the Apache-2.0 license
- Focused on correctness before performance claims

## License and provenance policy

Do not copy code from GPL, AGPL, SSPL, BUSL, proprietary, or unknown-license sources.

It is acceptable to independently implement algorithms described in papers, specifications, or documentation, provided the implementation is independently written and attributed where appropriate.

AI-assisted contributions are welcome, but contributors are responsible for ensuring generated code is original, tested, and compatible with this repository's license policy.

## Development priorities

ShardLoom prioritizes:

1. Correctness
2. License cleanliness
3. Reproducible benchmarks
4. Vortex-native execution
5. Standalone operation
6. Ecosystem-compatible outputs
