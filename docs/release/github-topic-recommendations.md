# GitHub Topic Recommendations

Recommended repository topics:

- `vortex`
- `columnar`
- `analytics-engine`
- `data-engine`

- `no-fallback`

These topics describe the project direction and discovery surface. They do not
change support claims: `local_vortex_analytics_v1` is the current scoped
workload-certified slice, while broad SQL, DataFrame, live/hybrid,
object-store, and Foundry support remain future or explicitly unsupported until
certified.
