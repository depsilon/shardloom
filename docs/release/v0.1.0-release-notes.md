# ShardLoom v0.1.0

ShardLoom v0.1.0 is the first approved technical-preview release sequence for the local
Vortex-first, no-fallback compute engine foundation.

## What Is Included

- Source checkout and local build workflow.
- `shardloom` Python package with the scoped Python client/context surface.
- `shardloom` CLI release artifact path for local status, smoke, capabilities, and certified local
  workflow commands.
- Local CSV, JSONL/NDJSON, flat JSON, generated rows, local Vortex, and selected local output/sink
  routes described in the public docs.
- Current local benchmark artifacts as scoped evidence only.
- Release evidence bundle with checksums, SBOM/provenance dry-run artifacts, and no-fallback
  package-channel guardrails.

## Release Boundaries

This release is not a production, performance-superiority, Spark-replacement, broad SQL/DataFrame,
object-store/lakehouse, distributed, live/hybrid, or Foundry production claim.

Package-channel publication follows this order:

1. GitHub v0.1.0 pre-release and attached source/package/evidence assets.
2. TestPyPI Trusted Publisher upload and clean install/uninstall/smoke proof.
3. PyPI Trusted Publisher upload after TestPyPI proof.
4. Homebrew tap formula against the immutable GitHub v0.1.0 source archive.

Every channel must preserve `fallback_attempted=false` and `external_engine_invoked=false`.
